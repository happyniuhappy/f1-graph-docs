# setCenter()

设置中心点

Syntax

setCenter(val: Array<Number>, isTransform: Boolean): void

Parameters

  • val: Array<Number>
    中心点坐标

  • isTransform: Boolean
    是否转换坐标系，默认为true，源坐标为EPSG:4326坐标
