# 动画参数对象

# 一、动画js引入（引入animation方式调用，使用SvgStacicView方式引用请查询SVG绘制组件）

import { animation, stopAnimation } from "./animation"

animation为动画调用方法，包含两个参数：features-执行动画的图形数组、opt-动画相关参数对象数组，包含动画类型type等，返回数据为动画标识对象ID。

stopAnimation为停止动画调用方法，包含一个参数：animationId-动画标识对象ID，无返回数据。

# 二、动画方法调用（引入animation方式调用，使用SvgStacicView方式引用请查询SVG绘制组件）

1、动画执行

图形平台数据接口返回当前图层的所有图元信息，实际项目中依据自身需求过滤得出需要执行动画的图形数组，调用animation方法即可获取当前动画标识ID。（建议保存当前执行动画标识ID）

var animationId = animation(cells, opt);

2、动画停止

调用停止动画方法，传入需要停止的动画标识ID，即可停止该类动画效果。

stopAnimation(animationId);

# 三、动画参数对象

opt参数为动画相关参数对象，包含动画类型及不同类型下动画效果（颜色、频率、范围等）的设置。

opt = {
    type: "flash",      //动画效果，flash-闪烁效果，scale-放大效果，trend-潮流动画效果
    color: "#777777",   //默认颜色灰色
    scope: 2,           //默认范围（闪烁范围、放大范围），2倍大小
    times: 0,           //默认次数（闪烁次数），0为indefinite
    duration: 4         //默认频率（闪烁频率），4为4秒执行1次
};
