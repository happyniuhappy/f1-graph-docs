# 属性

参数 说明 类型 可选值 默认值
graphData 初始化数据 Object - {graphs: [], layer: null}
symbolsDef 初始化图元定义 Object - null
olTansfromFactor 图形缩放 Number - 1
isOLShowName 是否显示名称 Boolean true/false false
isOLShowLabel 是否显示标签 Boolean true/false false
isShowSlider 是否显示放大缩小的按钮及滑块 Boolean true/false false
isShowThumbView 是否显示缩略图/鹰眼模块 Boolean true/false false
contextmenus 右键菜单列表定义 Object - {}

# 方法

方法名 说明 参数 返回值
load 读取图层数据 layerId-图层ID -
loadFromString 读取数据 svgContent-数据对象 -
getOLDataByLayerId 获取svg渲染内容 layerId-图层ID, callback-回调函数 -
findCellsByPops 查找图形 {opt}-查找配置项，如{id: "abc"} 检索的图形数组
cellPosition 将图形缩放到中心 cell-图形对象 -
cellAnimation 添加图形动画 cells-图形对象数组，opt-动画设置数组（默认为闪烁） 图形动画标识
getAnimationCells 获取已设置动画的图形数组 - 动画标识及包含的图形数组，如： {"animation1": ["cell1", "cell2"]}
cellStopAnimation 清除图形动画（依据图形标识） 图形数组 —
stopAnimation 清除图形动画（依据图形动画标识） 图形动画标识数组 —

# 事件

事件名称 说明 回调参数
ready 组件初始化完成后触发 SvgStaticView实例对象
cell:pointerclick 点击图形触发 { 实例对象, dom元素, x, y, 图形ID, 图形类别type }
blank:pointerclick 点击画布空白处触发 { 实例对象, dom元素, x, y }
cell:contextmenu 右键点击图形触发 { 实例对象, dom元素, x, y, 图形ID, 图形类别type }
blank:contextmenu 右键点击画布空白处触发 { 实例对象, dom元素, x, y }
