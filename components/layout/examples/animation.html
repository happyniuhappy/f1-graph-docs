# SVG绘制动画示例

# 设置动画

    1. 引入 SvgStaticView 组件，并读取成图数据，进行整图绘制
    1. 调用 findCellsByPops 方法，查找过滤需要进行动画设置的图形

注：业务系统可根据自己需要，同时设置图形的业务数据信息，例如：潮流流向、设备属性等

    1. 调用 cellAnimation 方法，设置图形动画，并获取动画标识

注：动画参数参考下一节

    1. 停止动画，调用 stopAnimation 或 cellStopAnimation

# 动画参数

动画调用方法 cellAnimation(cells, opts)

相关参数：

opts = [{
    type: "flash",      //动画效果，flash-闪烁效果，scale-放大效果，trend-潮流动画效果
    color: "#777777",   //默认颜色灰色
    scope: 2,           //默认范围（闪烁范围、放大范围），2倍大小
    times: 0,           //默认次数（闪烁次数），0为indefinite
    duration: 4         //默认频率（闪烁频率），4为4秒执行1次
}];
