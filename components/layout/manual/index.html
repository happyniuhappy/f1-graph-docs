# 自动成图操作

# 一、单线图成图

# 使用demo提供的文件上传方式

  1. 使用页面“导入数据”按钮，选择需要成图的json文件，提交到成图服务接口

# 使用三方工具

  1. 使用接口调用工具（例如：postman），将需要成图的json文件发送到成图服务接口

json文件格式示例

需注意使用post方式，send后即可得到成图是否成功的提示


  1. 接口说明

# 简要描述

  • 导入JSON文件自动成图功能接口

# 请求URL

  • http://xx.xx.xx.xx:xx/zuul/giscore/layout/poleLineNoPosition.do

# 请求方式

  • POST

# 参数

参数名 必选 类型 说明
filePath 是 MultipartFile JSON文件
websocketTid 否 string ws主题
websocketIid 否 string ws标识

# 返回示例

//普通请求返回
{"successful":true,"resultHint":"成图完成","resultValue":"ed1e77a8-edaf-4901-b32c-9d17235c67e2"}
{"successful":false,"resultHint":"成图失败，请检查数据","resultValue":""}

//ws请求
{"successful":true,"resultHint":"已发送异步请求，请等待返回结果","resultValue":""}
//ws返回
{"successful":true,"resultHint":"成图完成","resultValue":"ed1e77a8-edaf-4901-b32c-9d17235c67e2"}
{"successful":false,"resultHint":"成图失败，请检查数据","resultValue":""}

# 返回参数说明

参数名 类型 说明
successful boolean 操作结果
resultHint string 提示信息
resultValue string 返回数据（layerId）

# 二、成图静态浏览

  1. 打开静态浏览页面，在左侧列表中选择相应的图层，即可在页面上浏览。

静态浏览页面包括：图层列表、绘制区域、操作按钮、鹰眼模式等部分


  1. 选择相应图层后，自动进行图形绘制

可以使用鼠标工具，或点击缩放按钮，进行拖动缩放等操作 同时在鹰眼区域，显示当前区域在整个图形的位置

  1. 图形动画
  • 点击图形后，图形高亮显示

  • 检索到相应的图形后，动画显示

  • 其他动画效果包括潮流等。
