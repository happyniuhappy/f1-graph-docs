      # OLBar 工具栏

      地图工具栏是一组功能按钮的容器。

      提示

      示例中使用的字体图标请使用时自行导入，这里仅示例使用

      <template>
    <div style="height: 400px; width: 100%">
        <!-- 地图组件 -->
        <OLMap>
            <!-- 基础底图高德瓦片 -->
            <AMapTileLayer />
            <!-- 工具栏 -->
            <OLBar position="top-left" :toggleOne="true">
                <OLButton title="按钮-1" icon="maki-fuel"/>
                <OLButton title="按钮-2" icon="maki-marker_stroked"/>
            </OLBar>
        </OLMap>
    </div>
</template>

      # 属性

      参数 说明 类型 可选值 默认值
      position 工具条位置 String top/top-left/left/bottom-left/bottom/bottom-right/right/top-right -
      visible 工具条是否可见 Boolean true/false true
      className 控件css样式类名 String — —
      toggleOne 一次仅激活一个切换控件 Boolean true/false false
      group 是否是一个组 Boolean true/false false
      autoDeactivate 与子栏配合使用，以在停用顶级控件时禁用所有控件 Boolean true/false false

      # 事件

      事件名称 说明 回调参数
      ready 组件初始化完成后触发 ol工具栏对象
