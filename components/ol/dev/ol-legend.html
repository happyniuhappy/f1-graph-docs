      # OLLegnd属性

      参数 说明 类型 可选值 默认值
      className 控件css样式类名 String — —
      title 图例标题 String — 图例
      size 图例中符号的大小 Array — [40, 25]
      margin 符号边距的大小 Number — 10
      collapsed 指定在启动时是否应折叠属性 Boolean true/false true
      collapsible 指定是否可折叠 Boolean true/false true
      target 如果要使控件在地图视口之外呈现，请指定一个目标 String — null

      # OLLegndItem属性

      参数 说明 类型 可选值 默认值
      geoType 图例类型 String 'Point','LineString','Polygon','Circle' 'Point'
      properties 图例属性 Object — —
      radius 半径，单位：米 Number — 3

      # 事件

      事件名称 说明 回调参数
      ready 组件初始化完成后触发 feature对象
