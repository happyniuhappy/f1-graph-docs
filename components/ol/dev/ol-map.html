      # OLMap地图

      基本的地图组件，只包含一个基础地图

      # 基础地图

      增加一个外层容器，给出固定高度，如果没有外容器包裹自适应父组件宽高

      <template>
    <!-- 外层容器 -->
    <div style="height: 400px; width: 100%">
        <!-- 地图 -->
        <OLMap>
            <!-- 基础底图高德瓦片 -->
            <AMapTileLayer />
        </OLMap>
    </div>
</template>

      # 设置初始化中心位置

      google服务已经无法访问，改为高德地图

      <template>
  <!-- 外层容器 -->
  <div style="height: 400px; width: 100%">
    <!-- 地图 -->
    <OLMap :center="center">
        <!-- google瓦片图层 -->
        <!-- <GoogleTileLayer/> -->
        <AMapTileLayer />
    </OLMap>
  </div>
</template>
<script>
    export default {
        data() {
            return {
                // 中恒博瑞
                center: [116.302778, 40.050833]
            }
        }
    }
</script>

      # 点击事件

      地图单击事件

      <template>
    <!-- 外层容器 -->
    <div style="height: 400px; width: 100%">
        <!-- 地图组件 -->
        <OLMap :zoom="false" :center="initCenter" @singleclick="onClick">
            <!-- 基础底图高德瓦片 -->
            <AMapTileLayer></AMapTileLayer>
        </OLMap>
    </div>
</template>

<script>
export default {
    data: function() {
        return {
            initCenter: [103.778611, 36.066111]
        };
    },

    methods: {
        onClick: function(evt) {
            alert("点击地图的像素位置："+evt.pixel);
        }
    }
};
</script>

      # 属性

      参数 说明 类型 可选值 默认值
      fitp 是否计算父容器的宽高，以适应父容器 Boolean true/false true
      center 地图中心位置 Array - [116.302778, 40.050833]
      zoomLevel 地图缩放层级 Number 1-18级 13
      maxZoom 地图最大缩放层级 Number - 18
      minZoom 地图最小缩放层级 Number - 0
      zoom 是否显示放大缩小地图的按钮 Boolean true/false false
      zoomSlider 是否显示控制地图放大缩小的滑块 Boolean true/false false
      rotate 是否可以旋转地图 Boolean true/false false
      mousePosition 是否显示当前鼠标的坐标位置 Boolean true/false false
      scaleLine 是否显示比例尺 Boolean true/false false

      # 方法

      方法名 说明 参数
      fit 地图缩放适应指定的范围 (geometryOrExtent, opt_options)，1.范围或几何形状对象 2.可选，配置选项，详细内容参照OpenLayers API
      moveCenter 移动到指定位置 接受一个参数，{center：坐标， zoom：缩放层级（选填），duration：持续时间}
      getMap 获取地图对象 —
      getZoom 获取地图当前缩放层级 —
      setZoomLevel 设置地图当前缩放层级 (zoom)，1至18数值
      addLayer 添加图层 接受一个参数，ol/Layer实例对象
      addControl 添加控件 接受一个参数，ol/Overlay实例对象
      addInteraction 添加交互操作控件 接受一个参数，ol/Interaction实例对象
      addOverlay 添加地图覆盖物 接受一个参数，ol/Overlay实例对象
      clearOverlays 清除地图覆盖物 —
      removeOverlayById 清除地图覆盖物根据指定的覆盖物对象id 接受一个参数， 覆盖物对象id
      transformFn 将无偏移的经纬度转换为偏移后的经纬度 (geoType, coordinates)接受2个参数， 1.图形的几何类型 2.经纬度坐标，数组对象
      unTransformFn 将偏移的经纬度转换为无偏移的经纬度 (geoType, coordinates)接受2个参数， 1.图形的几何类型 2.经纬度坐标，数组对象
