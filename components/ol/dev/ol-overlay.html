      # OLOverlay 叠加层

      地图矢量图层上显示叠加层控件

      Menu

      ol.control.Overlay can be used to display a menu or information on the top of the map.

      test
      <template>
<div style="height: 400px; width: 100%">
    <!-- 地图组件 -->
    <OLMap>
        <!-- 基础底图高德瓦片 -->
        <AMapTileLayer />
        <OLBar :position="barPosition">
        <OLToggle className="toggle-overlay" icon="icon-gis-layer-open">
            <OLOverlay className="menu">
                <div class="menu">
                    <h1>Menu</h1>
                    <p style="border-bottom:1px solid #999;">
                        <i>ol.control.Overlay</i> can be used to display a menu or information on the top of the map.
                    </p>
                    <div class="data">test</div>
                </div>
            </OLOverlay>
        </OLToggle>
        </OLBar>
    </OLMap>
  </div>
</template>

<script>
export default {
    data: function(){
        return {
            barPosition: 'top-left'
        }
    }
}
</script>
