      # OLTileLayer瓦片图层

      支持Google、AMap（高德）、BMap（百度）瓦片图层

      警告

      由于国情因素，Google瓦片可能被墙，无法连接。开发使用时请根据实际情况酌情选择底图瓦片

      # 扩展在线瓦片底图

      连接google卫星影像瓦片（偏移）

      <template>
    <!-- 外层容器 -->
    <div style="height: 400px; width: 100%">
        <!-- 地图 -->
        <OLMap>
            <!-- 瓦片图层 -->
            <OLTileLayer :url="imageTileUrl"/>
        </OLMap>
    </div>
</template>
<script>
export default {
    data() {
        return {
            // google卫星影像瓦片（偏移）
            imageTileUrl: "http://mt2.google.cn/vt/lyrs=s@258000000&hl=zh-CN&gl=CN&src=app&s=Ga&x={x}&y={y}&z={z}"
        };
    }
}
</script>

      # 离线瓦片

      离线瓦片需求提前利用下载工具进行下载，并利用服务工具如Nginx、Tomcat等发布出来，供内网环境访问。

      <template>
    <!-- 地图 -->
    <OLMap>
        <!-- 瓦片图层 -->
        <OLTileLayer :url="offLineTileUrl"/>
    </OLMap>
</template>
<script>
export default {
    data() {
        return {
            // 下载的google卫星影像瓦片（偏移），发布到内网的离线瓦片地址
            offLineTileUrl: "http://192.168.131.111:8088/tiles/googlemaps/satellite/{z}/{x}/{y}.jpg"
        };
    }
}
</script>
