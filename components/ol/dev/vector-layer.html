      # OLVectorLayer矢量图层

      矢量图层是图形容器，地图数据展示是一个个图层叠加形成。

      <template>
    <!-- 外层容器 -->
    <div style="height: 400px; width: 100%">
        <OLMap>
            <AMapTileLayer/>
            <!-- 矢量图层组件 -->
            <OLVectorLayer guid_="test1" layerId="test-layer-1" layerName="测试图层" layerType="1800101"></OLVectorLayer>
        </OLMap>
    </div>
</template>
<script>
export default {
    data() {
        return {
            message: 'Hello Vue'
        }
    }
}
</script>

      # 属性

      参数 说明 类型 可选值 默认值
      guid_ 图层唯一标识 String — —
      layerId 图层id String — —
      layerName 图层名称 String — —
      layerType 图层类型 String — —
      title 图层标题 String — —
      visible 是否可见 Boolean false/true true
      zIndex 层级索引，同style的z-index作用相同 Number — 10
      maxResolution 最大解析率 Number — —
      minResolution 最小解析率 Number — —
      opacity 透明度 Number 0到1 1
      source 数据源，ol/source/Vector实例对象 Object — —
      declutter 整理图像和文字。 整理应用于所有图像和文本样式，并且优先级由样式的z-index定义。 较低的z-index表示较高的优先级。 Boolean — false
      updateWhileAnimating 设置为true且renderMode为vector时，将在动画过程中重新创建功能批次。 这意味着将不会显示任何裁剪的矢量，但是此设置会对大量矢量数据产生性能影响。 设置为false时，如果没有动画处于活动状态，则将重新创建批处理。 Boolean false/true false
      silentSvg 是否静态渲染svg定义的符号。 Boolean false/true true
      isSymbolsWithMapZoom 图元是否随着地图缩放而缩放。 Boolean false/true true
      initSymbolsZoomLevel 初始的图元放大的比率。默认值：1/Math.pow(2, 14)，14代表是缩放层级，表示在14缩放层级下图元的比例为1比1 Number 1到18 14
      properties 初始的图元放大的比率。 Object — —

      # 方法

      方法名 说明 参数
      setSource 设置数据源 接受一个参数，ol/source/Vector实例对象
      getMap 获取地图对象 -
      getExtent 获取范围 -
      fit 地图范围适应图层数据范围 接受一个参数，可选，例如：{duration: 500, delay: 250}
      clear 清除图形数据 -

      # 事件

      事件名称 说明 回调参数
      ready 组件初始化完成后触发 ol/layer/Vector实例对象
