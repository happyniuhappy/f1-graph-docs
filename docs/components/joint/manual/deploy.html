# 2.部署

图形平台服务端依赖于f14.0微服务平台，部署之前必须已经部署f1微服务，这里不在重复部署步骤，详细参见f1微服务部署手册。 图形平台主要包含：前端微服务、服务端微服务。

# 2.1配置参数

# 2.1.1客户端参数

  • 客户端网络要求

局域网环境；

在TCP/IP协议下的固定或动态IP地址；

  • 客户端硬件要求

硬盘：50M以上剩余空间；

内存：128M以上；

CPU：1.5GHz以上

  • 客户端软件要求

微软视窗系列操作系统（Windows2000以上）/Andriod4+/IOS

浏览器IE11+/Chrome/Safari

# 2.1.2服务器端参数

  • 服务器网络要求

局域网环境；能连接到因特网（可选）；

在TCP/IP协议下的固定IP地址，或者动态IP加动态域名；

  • 服务器硬件要求

硬盘：5G以上剩余空间；

内存：4GB以上；

CPU：奔腾PentiumⅣ系列，主频1.5GHz以上，

在线人数达到500人以上，上建议使用双CPU服务器；

磁盘阵列/光盘库/磁带机（可选）；

  • 服务器软件要求

微软视窗系统环境（Windows2000 Server以上）；

# 2.1.3技术参数

1) 平均无故障时间（MTBF）：≥4760小时；

2) 画面调用时间：<10秒；

3) 画面实时数据刷新时间：5秒～30秒；

4) 实时数据查询响应时间：<3秒；

5) 历史数据查询响应时间：<10秒；

6) 正常情况下CPU负载：≤20%（一分钟均值）

7) 事故情况下CPU负载：≤50%

8) 系统可用率： ≥99.9%

# 2.2服务端微服务部署

进入f1微服务部署的根目录下，使用的步骤如下：

第一步：f1-giscore-service、f1-gisadvance-service两个文件夹放入微服根目录下。

第二步：lib文件夹是图形引用jar包，需要与平台部署的lib合并。

第三步：打开bootstrap.properties、application.properties文件，修改相关配置地址及数据库路径。

第三步：进入图形微服务目录下，打开命令行窗口，输入"java -jar f1-giscore-service-3.0.0-RELEASE.jar"（linux：java -jar f1-giscore-service-3.0.0-RELEASE.jar &）启动命令。

注：使用服务的默认配置文件，不要使用平台默认的通用配置文件。

# 2.3 前端微服务部署

依赖：Vue2.6.10以上版本。

# 2.3.1 安装模块

使用joinbright的npm库进行安装。

// 安装nrm，管理npm镜像地址
npm install nrm -g
// 添加私服地址，名称为joinbright
nrm add joinbright http://192.168.1.172:8081/repository/npm-all/
// 切换镜像到私服
nrm use joinbright
// 插入图形组件库
npm install f1-graph-vue

# 2.3.1 Vue配置

vue工程目录下，在vue.config.js内添加如下配置，能够解析joint依赖：

// webpack相关配置
chainWebpack: (config) => {
    if(process.env.NODE_ENV === 'production'){
        // 生产环境下使用压缩文件，不在进行loader处理
        config.resolve.alias.set('joint$', 'jointjs');
    } else {
        // 开发环境使用joint源码引用
        config.resolve.alias.set('joint$', 'jointjs/joint.mjs');
    }
}
