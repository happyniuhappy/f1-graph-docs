# 管理维护工具

# 1、功能介绍

管理维护工具作为图形平台支撑模块，负责为图形平台进行管理维护功能，包括系统主菜单、图元分组、图元列表、系统下拉菜单数据、系统属性数据、系统工具栏、系统右键菜单管理，另外对于电力资源数据，可以进行电力设备电压等级、电力设备开关状态进行管理。

# 2、各模块说明

# 2.1、主菜单管理

# 2.1.1、界面

主菜单管理负责图形平台顶部主菜单内容配置管理，主菜单为两级结构，左侧目录树为一级菜单，右侧表格默认显示一级菜单数据，左侧目录树点击某一级菜单，右侧显示该一级菜单下的二级菜单。

查看模式下，可以查看主菜单模块的基本信息，一级Click方法代码查看，如果没有click方法，代码编辑框为空。

# 2.1.2、新增

点击右侧表格 “新增” 按钮，打开新增浮动框。

填写信息后保存即可。

# 2.1.3、Click方法设置

新增或修改菜单模块时，可以设置该菜单模块的Click方法，点击编辑浮动框中的 “Click方法”的“编辑” 按钮，可以打开代码编辑框。

该click方法设置结果为：在图形平台（目前主要为编辑模式）中，主菜单按钮点击后执行的方法，其代码中this指向MainMenu的VUE对象。

删除主菜单该模块
