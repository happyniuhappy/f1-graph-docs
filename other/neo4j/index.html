# 简介

在我们周围的世界中，互联信息无处不在。Neo4j 旨在高效地存储、处理和查询数据模型中高度连接的数据。借助强大而灵活的数据模型，您可以在不损失丰富性的情况下表示真实世界的可变结构信息。

# Docker 安装

  1. 拉取镜像
docker pull neo4j:3.5.28
  1. 启动容器
docker run -p7474:7474 -p7687:7687 -e NEO4J_AUTH=neo4j/s3cr3t neo4j

首次登录

  • 初始化默认用户neo4j、密码neo4j
  • 首次登陆neo4j会提示修改密码，如果需要要禁用，请设置环境变量：NEO4J_AUTH=none
  • 7474端口是http访问，7687端口是bolt协议访问
  1. 在浏览器输入 http://服务IP:7474 打开登陆页面。例如
http://192.168.1.67:7474/browser/
  1. 点击Connect URL下来列表选择“bolt://”，输入ip:bolt协议端口。例如
bolt://192.168.1.67:7687

然后输入用户密码，点击connect按钮进行登录。
首次登陆需要修改密码

更多详细内容，请查看官方文档 (opens new window)
