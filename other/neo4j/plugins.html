# 如何安装插件

  1. 插件存放路径：neo4j安装目录\plugins
  2. 将下载的插件，也就是一个jar包放入plugins目录内，重启neo4j。

# APOC

APOC 是 Neo4j 的附加库，它为neo4j提供了数百个存储过程和函数，并添加了许多有用的功能。
可帮助完成数据集成、图形算法或数据转换等领域的许多不同任务。

# 使用案例

  • 列出空间插件所有的存储过程信息
call spatial.procedures()
  • 创建一个普通节点，并根据该节点的wkt属性将该节点添加到地理空间图层中，该节点成为一个空间节点
create (n:Feature {wkt: "[105.380859375,31.57853542647338]"}) 
with n as n
call spatial.addNode("geom", n) yield node return node
  • 查询距离给定位置最近的节点
call spatial.closest("geom", point({longitude: 105.380859375, latitude:31.57853542647338}), 1) yield node return node
  • 删除空间节点
// 从地理空间图层中删除一个地理空间节点
match (n:Feature) with n call spatial.removeNodes("geom", n)

// 删除多个地理空间节点，并返回删除数量
match (n:Feature) with [n] as s call spatial.removeNodes("geom",s) yield count return count
