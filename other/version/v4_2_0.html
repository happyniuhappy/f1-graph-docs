# V4.2.0

图形本版本主要新增地图常用功能，主要包含数据展示、搜索查询、编辑绘制与数据存储。

# 功能列表

# OLTileLayer瓦片底图

  • Google瓦片（谷歌在线、离线瓦片）
  • Amap瓦片（高德地图在线、离线瓦片）
  • Bmap瓦片（百度地图在线、离线瓦片）

# VectorLayer矢量图层

  • 按地图不同的zoomLevel缩放层级显示、隐藏
  • 支持OLSource矢量数据源，是矢量数据源的父容器

# OLSource数据源

  • 展示符合GeoJSON数据地图数据
  • 解析连接拓扑关系
  • 图形数据的检索、添加、删除

# Feature矢量要素

  • 支持字体图标图元符号定义与渲染
  • 支持SVG图元符号定义与渲染
  • 仅支持Point、LineString、Polygon、Circle几何图形
  • 显示、隐藏
  • 位置坐标设置
  • 渲染样式设置
  • 位置缓冲范围查询
  • 支持GCJ02、WGS84、BD02之间的坐标转换
  • 按地图不同的zoomLevel缩放层级展示

# OLControl地图控件

  • OLBar工具栏，操作按钮容器，可嵌套使用并支持子工具栏
  • OLButton功能按钮，单击击操作，显示内容仅支持字体图标
  • OLTextButton功能按钮，单击操作，按钮显示内容仅支持文字
  • OLToggle状态切换按钮，支持激活、失效等地图交互编辑操作功能
  • OLLayerSwitcher图层管理功能
  • OLSearchFeature矢量图形要素搜索定位，仅支持地图中加载的数据检索
  • OLOverlay地图中展示的地标、信息框、提示框等覆盖物
  • OLOverview鹰眼地图
  • OLLegend图例控件
  • OLToggleSnap捕捉功能、编辑时捕捉到给定图形的顶点、边坐标
  • OLSave保存图形数据

# OLInteraction地图交互操作

  • 绘制Point、LineString、Polygon图形要素的常规功能
  • 绘制带有连接拓扑关系的LineString几何图形要素
  • 捕捉带有连接热点（锚点）的SVG图元符号
  • 图形要素选中
  • 距离测量
  • buffer缓冲
  • 图形要素复制、粘贴
  • 图形要素几何图形顶点、边修改编辑
  • 图形要素offset（复制并偏移）操作
  • 图形要素几何形状变换操作
  • 图形要素位置平移操作
  • LineString图形与Point图形连接关系的编辑操作

# OLAnimation动画效果

  • Pulse脉冲效果
  • 路径方向箭头指示
  • 图形要素隐藏、显示时图形飞入、飞出效果
  • 图形闪烁
