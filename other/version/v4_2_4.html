# V4.2.4

# 功能完善清单

# 地图：

  • 默认Google瓦片地图替换为AMap高德地图
  • GeoJSON数据导入导出
  • dxf（CAD）格式导出
  • png格式导出
  • 缩放地图，图元符号大小随缩放级别动态缩放
  • 支持电缆通道图元符号渲染
  • 新增开始编辑、停止编辑控件
  • 新增新建图层、打开图层功能
  • 地图编辑后保存功能
  • 按钮控件支持svg、png、base64格式的图标显示

# 组态图：

  • websocket关闭接口暴露
  • 删除工程后清空画布
  • 保存工程只提示一遍，已保存未修改不再提示
  • 读取多个图层数据接口完善
  • 增加图形组合功能
  • 输入坐标偏移量移动选中图形
  • 增加拉框放大功能
  • 增加按属性查询功能
  • 用户控件共享使用
  • 图形绘制保存和发布保存增加状态识别标识
  • 属性编辑器下拉列表支持服务端配置
  • 表单验证提示信息异常问题修复
  • 切换tab页后画布位置偏移问题修复
  • 运行监控页面，数据列表增加“刷新”按钮，可刷新数据列表
  • 屏蔽后台Expction抛出的异常提示信息返回到前端

# 图元设计器：

  • 升级到Svg-Edit到6.0.0版本
  • 点击一次撤销返回多步操作问题修复
  • 旋转后的图形对齐不正确问题修复
